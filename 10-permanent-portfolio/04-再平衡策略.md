# 第四章：再平衡策略

> 再平衡是永久投资组合的"维护程序"，正确执行再平衡才能维持策略有效性。

---

## 什么是再平衡？

由于各资产价格变动，原本25/25/25/25的比例会偏离。**再平衡就是将偏离的比例重新调整回25%。**

### 示例

```
初始状态（2023年初）：
股票 25% | 长债 25% | 黄金 25% | 现金 25%

一年后（股票大涨，黄金小涨）：
股票 35% | 长债 20% | 黄金 28% | 现金 17%
         ↑需要卖出   ↑需要买入  ↑需要卖出  ↑需要买入

再平衡操作：
1. 卖出多余股票（35% → 25%，卖出10%）
2. 卖出多余黄金（28% → 25%，卖出3%）
3. 买入长债（20% → 25%，买入5%）
4. 买入现金（17% → 25%，买入8%）
```

---

## 再平衡触发条件：两种方式

### 方式一：时间触发（日历再平衡）

**每年固定日期检查并再平衡一次。**

```
建议时间：每年1月第一个交易日
原因：
- 上一年税务已结清
- 新年开始"重置"心理账户
- 避免年底换仓的税务损失
```

**执行流程：**
1. 打开投资账户，记录各资产当前市值
2. 计算总市值
3. 计算各资产偏离情况
4. 若任一资产偏离超过5%，执行再平衡
5. 若所有资产偏离<5%，**不操作**

---

### 方式二：阈值触发（波段再平衡）

**当任一资产比例超出目标值±15个百分点时触发。**

哈里·布朗原始建议的阈值规则：
```
目标比例：25%
触发下限：25% - 15% = 10%（任一资产跌至10%以下时买入）
触发上限：25% + 15% = 40%（任一资产涨至40%以上时卖出）
```

**更常用的实用版本：**
```
触发阈值：任一资产偏离目标比例 ±10%（即超出或低于25%超过10个百分点）
即：低于15% 或 高于35% 时触发
```

---

### 两种方式的比较

| | 时间触发 | 阈值触发 |
|--|---------|---------|
| **操作频率** | 每年1次，固定 | 不定期，可能0次也可能多次 |
| **简单性** | ★★★★★ | ★★★☆☆ |
| **税务效率** | 可规划 | 不可预测 |
| **市场适应性** | 较差 | 较好 |
| **推荐人群** | 懒人投资者 | 关注成本的投资者 |

**我的建议：使用时间触发（每年一次）+ 阈值监控（极端情况才触发）。**

## 再平衡的具体操作

### 计算示例

```
当前总市值：120,000元

资产      当前市值    当前占比    目标占比    差额
股票       42,000元     35%        25%      -12,000元（需卖出）
长期国债   24,000元     20%        25%      +6,000元（需买入）
黄金       33,600元     28%        25%      -3,600元（需卖出）
现金       20,400元     17%        25%      +9,600元（需买入）
```

**操作步骤：**
1. 卖出股票ETF 12,000元
2. 卖出黄金ETF 3,600元
3. 合计获得现金 15,600元
4. 买入长债ETF 6,000元
5. 买入货币基金 9,600元

---

### 使用新增资金再平衡（免税技巧）

如果有新增投资资金（如年终奖），优先用新资金补充"低于目标"的资产，而不是卖出"高于目标"的资产：

```
场景：年终奖 20,000元准备投入

当前偏差：
- 股票偏高（多5,000元）
- 长债偏低（少8,000元）
- 现金偏低（少12,000元）

操作：
将20,000元全部投入偏低的资产：
- 长债ETF买入 8,000元
- 货币基金买入 12,000元

结果：无需卖出，无需缴税（中国资本利得税0%，但美股投资者避免了应税事件）
```

## 税务考量

### 中国大陆投资者

```
A股ETF交易：
- 买卖差价：暂不征收资本利得税（2024年现行政策）
- 分红/利息：代扣10%预提税
- 货币基金收益：免税

影响：中国投资者税务负担较轻，再平衡摩擦成本主要是交易佣金
```

### 美股投资者（美国税务居民）

```
短期资本利得（持有<1年）：按普通收入税率，最高37%
长期资本利得（持有≥1年）：0% / 15% / 20%

策略：
1. 持有每笔ETF超过1年后再卖出（用于再平衡）
2. 优先在IRA/401(k)退休账户内再平衡（免税）
3. 使用新增资金买入低配资产，减少卖出
4. 税损收割（Tax-Loss Harvesting）：年底卖出亏损头寸，买入类似ETF
```

### 海外华人（非美国税务居民）

```
W-8BEN持有人：
- 资本利得：美国不征税（由居住国征税）
- 股息：预提税30%（部分国家有税收协定可降至10-15%）

建议：
- 尽量投资不派息或少派息的ETF
- 咨询当地税务顾问
```

## 再平衡成本分析

每次再平衡的成本包括：

| 成本项目 | A股 | 美股 |
|----------|-----|------|
| 交易佣金 | 0.025-0.03% | $0（多数券商） |
| 买卖价差 | 约0.01-0.05% | 约0.01-0.02% |
| 税务成本 | 0%（暂免） | 视持有期 |
| **每次总成本** | **约0.1%** | **约0.05%** |

**年均成本（每年1次）：约 0.05-0.1%**

## 再平衡心理学：克服常见冲动

### 冲动一："股票涨了这么多，不应该继续持有吗？"

**答案：不。** 再平衡强制你"卖高买低"，这正是大多数散户做不到的事。

### 冲动二："黄金一直跌，我是不是配置错了？"

**答案：不。** 黄金可能连续5-10年跑输，但它在危机时发挥作用。2022年美股大跌30%时，黄金几乎持平，正是它发挥了保护作用。

### 冲动三："利率这么高，现金部分是不是应该更多？"

**答案：不。** 任何偏离25/25/25/25的操作都是在做择时预测，而择时几乎注定失败。坚守比例。

## 再平衡检查清单

```
✓ 每年1月第一周执行检查
✓ 记录各资产当前市值和占比
✓ 若任一资产偏离>10%，执行再平衡
✓ 优先使用新增资金（而非卖出）来平衡
✓ 确认再平衡后四类资产各约25%
✓ 记录操作日志（日期、买卖金额、原因）
✓ 检查ETF是否仍符合标准（规模、费率未变化）
```
