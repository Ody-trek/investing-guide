## 每个危机期间的表现

```
标普500：-45%
永久组合：+16%（3年累计）

原因：股票下跌，但黄金和债券大涨补偿了损失
```

### 2008 全球金融危机

```
标普500峰值到谷底：-55%
永久组合峰值到谷底：-13.8%

最差月份（2008年10月）：
- 标普500：-16.8%
- 永久组合：-5.1%

永久组合保护了86%的跌幅
```

### 2020 新冠暴跌

```
2020年2-3月暴跌：
- 标普500：-34%（峰值到谷底）
- 永久组合：-12%

全年收益：
- 标普500：+18.4%
- 永久组合：+8.0%
（暴跌后因股票大反弹，永久组合错过了部分反弹）
```

```
2022全年：
- 标普500：-18.1%

- 黄金（IAU）：-0.6%
- 短债（BIL）：+1.4%
- 永久组合：-9.3%

教训：2022年是永久组合的弱项年份
股票和债券同时大跌，罕见的"双杀"
```

## 最大回撤深度分析

```
永久投资组合历史最大回撤事件：

1. 2022年：-13.8%（通胀+加息）
   恢复时间：约18个月

2. 2013年：-8.3%（黄金崩溃）
   恢复时间：约12个月

3. 2008年：-7.9%（金融危机）
   恢复时间：约6个月

4. 1981年：-6.2%（沃尔克加息）
   恢复时间：约9个月

结论：最坏情况约亏损14%，且通常在2年内恢复
对比标普500最坏回撤-55%，恢复需要4-5年
```

## 风险因子分析

### 系统性风险（无法分散）

1. **货币体系崩溃**：美元/人民币严重贬值，需要实物资产对冲
2. **政府违约**：国债价值归零（历史上极少见，但非零概率）
3. **黄金禁令**：如1933年美国强制收购黄金（历史案例）
4. **通胀+经济衰退（滞胀）**：1970年代是最难熬的时期

### 永久组合的阿喀琉斯之踵

```
最弱情境：股债双杀（如2022年）
条件：通胀高企 + 经济仍在运行 + 黄金表现平淡

2022年表现分解：
股票   -18%  × 25% = -4.5%
长债   -30%  × 25% = -7.5%
黄金    -1%  × 25% = -0.3%
现金    +1%  × 25% = +0.3%
全年总计：-12%（约）→ 实际约-9.3%（有时间价值）
```

## 压力测试：极端情景

### 情景一：1970年代式滞胀重演

```
假设：通胀10%/年，GDP增长1%，持续5年

预期表现：
- 股票：年化 -5%（滞胀对股票负面）
- 长债：年化 -15%（高通胀摧毁长债）
- 黄金：年化 +25%（黄金大牛市）
- 现金：年化 +8%（高利率环境）

永久组合5年累计：约 +10-15%（名义），实际-30-40%

教训：滞胀是永久组合最弱的情景
```

### 情景二：日本式通缩萧条

```
假设：通缩2%/年，股市停滞，持续10年

预期表现：
- 股票：年化 0%
- 长债：年化 +10%（利率降到零，债券价值飙升）
- 黄金：年化 -5%（通缩不利黄金）
- 现金：年化 +2%（购买力上升）

永久组合10年累计：约 +70%（名义），+90%（实际）

教训：这种环境下永久组合反而很好
```

### 情景三：2008年金融危机重演（更严重版本）

```
假设：股票-60%，债券+30%，黄金持平，现金+5%

单年表现：
股票   -60%  × 25% = -15.0%
长债   +30%  × 25% = +7.5%
黄金    0%   × 25% = 0%
现金   +5%   × 25% = +1.25%
合计：-6.25%

结论：即使严重危机，亏损仅约6%
```

## 数据结论

1. 永久投资组合在过去52年中**87%的年份为正收益**
2. **最大单年亏损约-9%**，远低于股票市场
3. **最大回撤约-14%**，通常在18个月内恢复
4. **年化收益约7-8%**，跑赢通胀约4-5个百分点
5. 在经济危机中（2002年、2008年）表现显著优于纯股票
6. **弱点是股债双杀年份**（2022年），但仍好于纯股票或纯债券
