# 第三章：建仓操作流程

> 从零开始，一步一步建立你的永久投资组合。

---

## 建仓前的准备工作

### 第一步：确定投资金额

**最低建议金额：**

| 账户类型 | 建议最低金额 | 原因 |
|----------|-------------|------|
| A股账户 | 5万元人民币 | 四类资产各约1.25万，ETF最小1手 |
| 港股账户 | 5万港元 | 部分ETF手数较大 |
| 美股账户 | 5000美元 | 零股交易普及，门槛低 |

**资金原则：**
- 只用**闲置资金**，不用生活备用金（建议留6个月生活费在账户外）
- 不用贷款或杠杆资金
- 这部分钱要能**至少5年不动用**

---

### 第二步：选择账户类型

#### A股投资者

```
推荐券商：
├── 华泰证券（涨乐财富通）— 佣金低，界面好
├── 东方财富（天天基金/证券）— ETF交易便捷
├── 国信证券 — 老牌稳健
└── 富途牛牛（港股+美股+A股）— 一站式

开户步骤：
1. 下载券商App
2. 身份证 + 银行卡（完成实名认证）
3. 签署风险测评（选择"中风险"或以上）
4. 出入金设置
预计时间：1-2个工作日
```

#### 美股/港股投资者

```
推荐券商：
├── 富途证券（Futu/moomoo）— 中国用户友好
├── 老虎证券（Tiger Brokers）— 中国用户友好
├── 盈透证券（Interactive Brokers）— 专业投资者首选
└── Charles Schwab — 美国本土，无佣金

开户步骤：
1. 准备护照（国际账户需要）
2. 填写W-8BEN表格（非美国公民税务申报）
3. 完成KYC（身份验证，通常需要1-3个工作日）
4. 入金（银行电汇或支付宝/微信，视券商而定）
```

---

### 第三步：计算各资产金额

以 **10万元人民币** 为例：

```
总资金：100,000 元

股票（25%）：25,000 元 → 购买沪深300ETF（510300）
长期国债（25%）：25,000 元 → 购买超长债ETF（511090）
黄金（25%）：25,000 元 → 购买黄金ETF（518880）
现金（25%）：25,000 元 → 货币基金（余额宝等）
```

以 **10,000 美元** 为例（美股）：

```
总资金：$10,000

股票（25%）：$2,500 → 买入 VOO（约 X 股，或零股）
长期国债（25%）：$2,500 → 买入 TLT
黄金（25%）：$2,500 → 买入 IAU
现金（25%）：$2,500 → 买入 BIL 或存入货币市场账户
```

## 建仓操作：手把手步骤

### A股建仓流程（以10万元为例）

#### 股票部分：买入沪深300ETF（510300）

```
操作步骤：
1. 打开证券App → 搜索"510300"
2. 点击"买入"
3. 选择"限价单"（推荐）或"市价单"
4. 金额换算：
   - 当前价格（如4.0元/份），25000元 ÷ 4.0 = 6250份
   - ETF最小交易单位100份，6250份 → 买入6200份（向下取整到100的整数倍）
5. 确认委托 → 成交

注意：交易时间为工作日 9:30-11:30 / 13:00-15:00
```

#### 长期国债部分：买入超长债ETF（511090）

```
操作步骤（同上）：
1. 搜索"511090"
2. 买入金额约25000元对应份数
3. 债券ETF流动性较股票ETF略低，建议用限价单

债券ETF净值通常在100元左右，波动较小
```

#### 黄金部分：买入黄金ETF（518880）

```
操作步骤：
1. 搜索"518880"（或"黄金ETF"）
2. 当前价格约3.5-4.5元/份（随金价波动）
3. 25000元 ÷ 当前价格 = 所需份数（取整到100的倍数）
4. 限价单买入
```

#### 现金部分：购买货币基金

```
操作步骤（以余额宝为例）：
方法一：支付宝 → 余额宝 → 转入25000元
方法二：券商App → 理财/货币基金 → 选R1产品 → 申购
方法三：国债逆回购（GC001）
  - 搜索"204001"（上交所）或"131810"（深交所）
  - 注意：逆回购T+1到期，到期自动返回账户
  - 利率随市场波动，一般2-3%/年
```

### 美股建仓流程（以$10,000为例）

#### 步骤一：资金到位

```
入金方式（以富途为例）：
1. 富途App → 资金 → 入金
2. 选择"银行转账"或"FPS/支付宝"
3. 按照指示完成转账
4. 一般T+1到账（美元）
```

#### 步骤二：逐一买入四类ETF

```
买入VOO（股票，$2,500）：
1. 搜索"VOO"
2. 点击"Buy"
3. Order Type选"Limit Order"（限价）
4. 设置价格：当前价±0.1%（接近市价）
5. 数量：$2500 ÷ 当前价（如$480）≈ 5.2股
   → 如支持零股（fractional shares）：输入$2500金额直接买入
   → 如不支持零股：买入5股（约$2400），余额留现金
6. 确认下单

重复以上步骤买入TLT、IAU、BIL
```

#### 步骤三：确认持仓

```
买入完成后检查：
✓ 四类资产都已持有
✓ 各占比接近25%（允许±2%误差）
✓ 记录首次建仓日期和价格（用于后续再平衡参考）
```

## 建仓时机的选择

### 常见问题：现在是好时机吗？

**华尔街的答案：对于永久投资组合，几乎任何时候都是好时机。**

原因：
1. 永久投资组合设计上不依赖市场时机
2. 尝试"择时"往往会适得其反
3. 市场高点买入，若持有5年以上，历史上几乎总是盈利的

### 建议策略：分批建仓

如果心理上担心"买在高点"，可以分批：

```
分3-6个月建仓：
第1个月：投入总计划金额的25%
第2个月：投入25%
第3个月：投入25%
第4个月：投入25%

优势：降低单一时点风险
劣势：历史数据显示，一次性投入约2/3概率优于分批
```

**个人建议：若金额<50万，一次性建仓；若>50万，分3个月建仓。**

## 常见建仓错误

| 错误 | 危害 | 正确做法 |
|------|------|----------|
| 等市场下跌再买 | 永远等不到"完美低点" | 确定目标后尽快建仓 |
| 用短期债券代替长期国债 | 失去通缩对冲效果 | 坚持20年+期限 |
| 买黄金矿股代替黄金 | 相关性不同，失效 | 买实物黄金ETF |
| 忘记现金部分 | 衰退期无缓冲 | 四个部分缺一不可 |
| 随意调整比例 | 破坏策略逻辑 | 严格25/25/25/25 |
