# 第七章：常见误区与FAQ

> 来自真实投资者的问题，来自华尔街的解答。

---

## 常见误区

### 误区一："永久投资组合可以保证不亏钱"

**错误。**

- 历史上有7个亏损年份（约13%的年份亏损）
- 2022年亏损约9.3%
- 最大单次回撤约14%

**正确理解：** 它是**亏损概率很低、亏损幅度很小**的策略，不是无风险保证。

---

### 误区二："我用3年期国债代替30年期，效果一样"

**完全错误，这是最常见的致命错误。**

```
原因分析：
3年期国债对利率变动的敏感度（久期）≈ 3
30年期国债的久期 ≈ 20

当利率下降1%：
3年期国债：涨约 3%
30年期国债：涨约 20%

在经济衰退/通缩期，央行大幅降息，长期国债的上涨
才能有效抵消股票的下跌。短期债券根本做不到。
```

**必须使用20年以上的长期国债。**

---

### 误区三："既然股票长期收益最高，我把股票比例提高到50%"

**破坏了策略的核心逻辑。**

- 提高股票比例意味着在经济繁荣期赚更多
- 但在熊市（如2000-2002、2008年）亏损会大幅增加
- 永久投资组合的核心是**均衡**，任何偏离都改变了策略性质

若你能承受更大波动追求更高收益，应该考虑别的策略，而不是修改永久组合。

---

### 误区四："买黄金股票（金矿ETF如GDX）来代替黄金"

**错误。黄金矿股 ≠ 黄金。**

```
黄金矿股的特性：
- 本质是股票，受整体股市影响
- 包含管理风险、运营成本等
- 与标普500相关性约0.5（高于实物黄金的0.1）

在2008年金融危机：
实物黄金：+5%（全年）
GDX（黄金矿ETF）：-26%

在危机时，黄金矿股跟随股票下跌，而非上涨。
这完全违背了持有黄金的目的。
```

---

### 误区五："每个季度再平衡一次更好"

**不对，过度再平衡损害收益。**

```
再平衡成本分析（每次约0.1%）：
季度再平衡：每年4次 × 0.1% = 0.4%/年的摩擦成本
年度再平衡：每年1次 × 0.1% = 0.1%/年的摩擦成本

历史研究显示：
年度再平衡 vs 季度再平衡，年化收益差别<0.2%
但季度再平衡的成本是年度的4倍

结论：年度再平衡是最优频率（或使用阈值触发）
```

---

### 误区六："市场崩溃时我应该全部转入现金"

**这是市场择时，不是永久投资组合。**

一旦开始"感觉不对就跑"，就完全放弃了策略的纪律性。历史数据显示：
- 业余投资者尝试择时，平均每年少赚约1.5-2%
- "感觉崩溃要来"时往往是最不该离场的时候（2020年3月就是例子）

**正确做法：无论市场如何，坚持比例，只做年度再平衡。**

---

### 误区七："A股的超长债和美国TLT效果一样"

**不完全一样，但方向正确。**

```
差异：
1. 久期：美国TLT久期约17-18年，中国超长债ETF约15年
2. 流动性：TLT日均成交额约30亿美元，中国超长债ETF约5亿元
3. 利率环境：中美利率周期不同步

影响：A股超长债的通缩对冲效果略弱于TLT，但仍然有效。
在无法配置美国国债时，超长债ETF是最好的替代。
```

## FAQ 常见问题

### Q1：我只有2万元，可以做永久投资组合吗？

**可以，但建议先用A股方案。**

```
2万元分配：
- 股票：5,000元 → 沪深300ETF（约1250份）
- 长债：5,000元 → 超长债ETF（约50份）
- 黄金：5,000元 → 黄金ETF（约1250份）
- 现金：5,000元 → 货币基金

门槛很低，ETF最小购买100份，以上均可实现。
```

---

### Q2：我已经60岁了，还适合用永久投资组合吗？

**非常适合，甚至比年轻人更适合。**

- 退休后资产保全比增值更重要
- 永久组合的低回撤特性非常适合退休阶段
- 无需主动管理，每年只需操作一次
- 建议略微提高现金/短债比例至30-35%，降低股票至20-15%

---

### Q3：现在通胀这么高，是不是应该提高黄金比例到50%？

**不要这样做。**

原因：
1. 通胀可能已经反映在金价中（市场提前预判）
2. 如果通胀被快速控制（如2022-2023年），黄金反而下跌
3. 你无法确定通胀会持续多久

**永久投资组合的意义就是"不需要预测"。增加黄金比例意味着你在预测通胀将持续，这是在做择时。**

---

### Q4：比特币可以替代黄金吗？

**目前不建议，但可以少量补充（不超过5%）。**

```
比特币 vs 黄金：
优点：
+ 供应上限2100万枚，通缩属性更强
+ 数字时代的"数字黄金"叙事
+ 历史上极端通胀下表现良好

缺点：
- 波动率极高（年化100%+），破坏组合稳定性
- 与风险资产相关性高（2022年随股票一起暴跌）
- 历史数据太短（<15年），不足以验证对冲效果
- 监管不确定性

结论：黄金有5000年历史证明，比特币没有。
作为永久组合的核心成分，不应超过5%。
```

---

### Q5：我应该买国内黄金还是COMEX黄金（美元定价）？

**基本一样，选择方便操作的即可。**

```
国内黄金ETF（518880等）：
- 人民币计价
- 追踪上海黄金交易所价格
- 与国际金价高度联动

国际黄金ETF（IAU/GLD等）：
- 美元计价
- 追踪COMEX/伦敦金价格
- 额外获得美元升值收益（若人民币贬值）

两者的黄金本身价值是相同的，差异仅在汇率层面。
境内投资者选国内黄金ETF即可，简单方便。
```

### Q6：永久投资组合和全天候组合（All Weather）有什么区别？

**目标类似，构建逻辑不同。**

```
永久投资组合（Harry Browne）：
- 四类资产各25%（等权重）
- 逻辑：四种经济状态均等可能

全天候组合（Ray Dalio/桥水）：
- 大约：股票30%、长债40%、中债15%、黄金7.5%、大宗商品7.5%
- 逻辑：按风险贡献均等（风险平价）
- 长债比例更高，是因为债券波动率低于股票

历史表现对比（美国，1990-2023）：
永久组合：年化约7.5%，波动率7%
全天候：年化约8.0%，波动率8%
相似，全天候略优，但也略复杂
```

### Q7：我持有这个组合，当股市大涨时看着别人赚大钱很难受，怎么办？

**这是永久投资组合的"代价"，必须接受。**

心理框架转换：

```
错误心态：
"今年股市涨30%，我只赚了8%，亏大了！"

正确心态：
"今年股市涨30%，我赚了8%，同时保护了我的资产，
如果明年股市跌30%，我可能只亏5%，
而追涨的人可能亏30%甚至离场。"
```

**关注绝对收益，不与股票指数比较。** 永久投资组合的基准不是标普500，而是"在任何经济环境下稳定正收益"。

### Q8：有没有更懒的方式执行永久投资组合？

**有！基金产品正在出现。**

```
美国市场：
- PERM ETF：直接购买，自动再平衡的永久组合ETF

中国市场（截至2024年）：
目前没有完全按照永久投资组合策略的公募基金，
但可以用"四合一"定投策略代替：
- 设置四个基金的自动定投（每月各投入1/4资金）
- 每年1月手动检查并再平衡一次

这样每年只需操作1次（再平衡），其余时间完全被动。
```

```
每年一次：
□ 记录各资产当前市值
□ 计算偏离程度
□ 若偏离>10%，执行再平衡
□ 更新持仓记录
□ 继续忽视市场噪音
```

## 最后的话

永久投资组合不是让你变得富有的秘密，而是让你**避免因错误决策而变穷**的工具。

它的真正价值在于：
- **不需要预测市场**
- **不需要专业知识**
- **不需要频繁操作**
- **在任何极端环境下都有保护**

巴菲特曾说："投资规则第一条：不要亏钱。第二条：永远记住第一条。"

永久投资组合正是这一哲学的最直接体现。
