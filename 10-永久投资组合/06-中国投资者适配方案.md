# 第六章：中国投资者适配方案

> 永久投资组合诞生于美国，但其原理适用于全球。本章专为中国大陆投资者提供本土化解决方案。

---

## 中国投资者面临的独特挑战

| 挑战 | 说明 |
|------|------|
| **资本管制** | 每年5万美元换汇额度限制 |
| **缺乏真正的长期国债ETF** | A股市场国债期限偏短 |
| **人民币汇率风险** | 配置美元资产需考虑汇率 |
| **政策不确定性** | 监管政策变化影响资产价格 |
| **税务体系差异** | A股暂免资本利得税 |

## 方案A：纯A股方案（最简单）

### 适合人群
- 无法或不便开通海外账户
- 投资金额较小（<50万元）
- 风险承受能力低，不想承担汇率风险

### 具体配置

```
纯A股永久投资组合
├── 股票（25%）：沪深300ETF（510300）
│   备选：中证500ETF（510500）或全A ETF
│
├── 长期国债（25%）：超长债ETF（511090）
│   备选：30年国债ETF（511150）
│   注意：久期比美国长债ETF略短，效果略打折
│
├── 黄金（25%）：黄金ETF（518880 或 159934）
│   备选：上海黄金交易所黄金积存
│
└── 现金（25%）：货币基金 + 国债逆回购
    推荐：余额宝/天弘基金（000198）
    补充：204001（GC001国债逆回购，利率更高）
```

### 历史回测（中国，2010-2023）

```
年化收益率：约 6.2%（人民币，名义）
年化波动率：约 8.5%
最大回撤：约 -12%（2015-2016年）
亏损年份：3/14

对比沪深300：
年化收益率：约 4.8%
年化波动率：约 22%
最大回撤：约 -47%（2015年牛市崩溃）
```

## 方案B：A股+QDII混合方案（推荐）

### 为什么要加入境外资产？

1. **分散地域风险**：中国股债黄金的相关性较高
2. **美元资产对冲**：人民币贬值时美元资产保值
3. **更完整的永久组合逻辑**：美国市场产品更成熟

### 具体配置

```
A股+QDII混合方案（总资金100%）

股票（25%）：
├── 12.5% — 沪深300ETF（510300）
└── 12.5% — 纳斯达克100 QDII（159632）或标普500 QDII（513500）

长期国债（25%）：
├── 15% — 超长债ETF（511090）
└── 10% — 美国国债QDII（场外基金，如工银瑞信美国国债）

黄金（25%）：
├── 15% — 黄金ETF（518880）
└── 10% — 伦敦金QDII 或 COMEX黄金QDII

现金（25%）：
└── 25% — 货币基金 + 国债逆回购（全部人民币）
```

### QDII产品注意事项

```
⚠️ QDII配额限制：
- 每家基金公司有QDII额度限制
- 热门QDII（如纳斯达克100）可能溢价交易
- 溢价率超过2%时，买入成本显著增加

检查溢价方法：
- 在交易软件查看"溢折价率"
- 若溢价>1%，等待额度放开或选替代产品
```

## 方案C：境内+境外双账户方案（最优）

### 适合人群
- 投资金额 > 50万元
- 可以开通港股/美股账户
- 愿意管理汇率风险

### 账户结构

```
总资产 100万元

境内账户（人民币，60%）：
├── 股票（15%）：沪深300ETF 15万
├── 长期国债（20%）：超长债ETF 20万
├── 黄金（15%）：黄金ETF 15万
└── 现金（10%）：货币基金 10万

境外账户（美元/港元，40%）：
├── 股票（10%）：VOO/3033.HK 10万
├── 长期国债（5%）：TLT/2823.HK 5万
├── 黄金（10%）：IAU/2840.HK 10万
└── 现金（15%）：BIL/港元货币基金 15万

总计：四类资产各约25%（跨境分配）
```

### 换汇策略

```
年度换汇计划：
每年使用5万美元额度（个人）= 约35万人民币

建议：
- 元旦换汇（全年额度重置）
- 分散在不同月份换汇（降低汇率时间集中风险）
- 使用银行"外汇宝"等工具锁定汇率

家庭可叠加：配偶5万美元 + 本人5万美元 = 10万美元/年
```

## 汇率风险管理

### 人民币贬值的影响

```
如果人民币贬值10%（如7.0→7.7），美元资产的人民币价值：
原来：$10,000 × 7.0 = 70,000元
贬值后：$10,000 × 7.7 = 77,000元（+10%）

持有美元资产本身就是对人民币贬值的对冲！
```

### 历史汇率参考

| 年份 | 美元/人民币 | 变化 |
|------|-----------|------|
| 2014 | 6.1 | |
| 2019 | 7.1 | -14% |
| 2020 | 6.5 | +8% |
| 2022 | 7.1 | -8.5% |
| 2023末 | 7.1 | 持平 |

**结论：长期看人民币有温和贬值压力，配置一定比例美元资产是合理的。**

## 特殊考虑：A股的独特性

### A股特点对永久组合的影响

| 特点 | 影响 | 应对策略 |
|------|------|----------|
| **政策市属性强** | 股票波动更剧烈 | 适当降低股票比例至20% |
| **债市与股市联动较高** | 分散效果略差 | 加入境外资产 |
| **黄金市场成熟** | 黄金ETF效果好 | 无需调整 |
| **货币市场发达** | 现金部分收益尚可 | 使用国债逆回购提升收益 |

### A股版本的调整建议

```
针对A股的微调方案：
股票：20%（降低，因A股波动更大）
长期国债：25%（保持）
黄金：25%（保持）
现金：30%（提高，利用高货币市场利率）
```

## 特别提示：不要忘记保险

永久投资组合管理的是**金融资产**，但完整的财富保全还需要：

```
财富保全金字塔：
    /永久投资组合\     ← 金融资产增值
   /  人寿+重疾险  \   ← 人身风险保障
  /  6个月生活备用金 \ ← 流动性储备
 /  自住房（若有）    \ ← 硬资产
/__________________________\
```

建议：
- 先配置充足的保险（寿险保额 = 年收入 × 10年）
- 保留6个月生活费作为应急金
- 然后再将剩余资金投入永久投资组合
